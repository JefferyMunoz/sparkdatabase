# spark 网页搜索工具

## 功能介绍

### 1. 输入 `网页快开设置` 命令进入设置界面，选择自己需要的搜索引擎
![image.png](https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f30965cb1c1a408f935adbc0856f6087~tplv-k3u1fbpfcp-watermark.image?)

### 2. 输入对应搜索引擎命令，进入搜索页面，输入完成想要搜索的关键词后，按回车即可搜索
![image.png](https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/de95f42d05a34813a46cf96fb6602d28~tplv-k3u1fbpfcp-watermark.image?)
